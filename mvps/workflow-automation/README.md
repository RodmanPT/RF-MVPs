# Workflow Automation MVP - SaaS RevOps

## Description
This MVP simulates the "Revenue operations automated end-to-end" solution from RemoveFriction's portfolio. It demonstrates how a rapid SaaS onboarding process can be automated across multiple fragmented tools (HubSpot, PostgreSQL, Stripe, OpenAI, Retool) with a single click, replacing manual handoffs.

## How to use it
Click the "Start Automated Onboarding for New Customer" button to watch the simulation of an end-to-end automated workflow spanning 6 different systems. It demonstrates the progress step-by-step.

## How to start it
1. From the root of the project, you can serve the directory using Python's built-in HTTP server:
   ```bash
   python3 -m http.server 8080
   ```
2. Navigate to `http://localhost:8080/mvps/workflow-automation/` in your web browser.
